First Background Color:

1st Background Color: Red

Color is based on RGB value: (255, 0, 0) or #FF0000. This color will be applied to the header area of the HTML.

2nd Background Color:

2nd Background Color: Dark-Grey-Blue

Color is based on RGB value: (54, 69, 79) or #36454F. This color will be used in the navigation tool bar.

Third Background Color

3rd Background Color: Light-Grey

Color is based on RGB value: (211, 211, 211) or #D3D3D3. This color will be used for the backgrounds of sections and articles.
